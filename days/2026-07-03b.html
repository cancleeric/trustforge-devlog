第五階段把「打開就懂、一眼看全局」做到位——讓使用者不必先做任何操作，一進站就看到跨幣種的信任全景，且全是真資料。

1. 首頁多幣信任總覽——一眼看懂五幣可信度

  • 一進站，首頁直接呈現 BTC / ETH / SOL / BNB / XRP 五幣的信任分總覽卡：每張卡標出該幣當前的信任分、市場方向、校準後的信心程度與決策狀態（正常結論／低信心／資料不足），以及資料時間戳。
  • 這些卡不是即時現算，而是背景排程定期產出的真實分析快照——每一張都是完整信任層 pipeline 跑出來的結果，不是預先寫死的樣本。開站零等待、卻是真資料。
  • 意義：市場情報產品最珍貴的是「全局視角」。與其一次只看一幣，不如先給你一個可信度全景，再讓你深入。

2. 預設就是真資料——差異化直接可見

  • 打開就是真實多源資料（新聞、鏈上、社群、官方公告、即時現價、群眾情緒、開發活躍），不需任何額外操作或參數。
  • 核心差異化「多源信任提煉」不再藏在後面——訪客第一次體驗，看到的就是真來源、真評分、可回溯，而且零成本（免大型模型的確定性信任演算法）。

3. 誠實面對「時間」——歷史基準 vs 此刻現價，分得清清楚楚

  • 每份分析都把官方歷史基準價（明確標注涵蓋到哪一天）與即時現價（真 API 回應、標注取得時間）並列呈現，各自帶時間戳。
  • 不再用「近兩週」這種會隨資料更新而逐漸失準的模糊措辭——你一眼分得清哪個是歷史基準、哪個是此刻的市場，不會被舊資料誤導。

4. 系統自己也透明

  • 背景抓取排程、各資料源的健康與鮮度、成本用量、運行版本，全部公開在系統狀態頁。多幣總覽的每一張卡都能追溯到它是哪一輪快照、什麼時間產出的。
